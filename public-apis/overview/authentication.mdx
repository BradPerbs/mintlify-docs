## API Key Authentication

All API requests require authentication using a workspace API key. You can generate API keys from your workspace settings.

### Obtaining an API Key

1. Navigate to **Settings** â†’ **API** in your dashboard
2. Click **Generate New Key**
3. Give your key a descriptive name (e.g., "Production API", "Development")
4. Copy and securely store your API key - it will only be shown once

<Warning>
  Keep your API key secret! Never expose it in client-side code or public repositories.
</Warning>

### Using Your API Key

You can authenticate requests using either the `Authorization` header or the `X-API-Key` header.

<Tabs>
  <Tab title="Authorization Header (Recommended)">
    ```bash
    curl -X GET "https://api.geogen.io/v1/entities" \
      -H "Authorization: Bearer wsk_your_api_key_here"
    ```
  </Tab>
  <Tab title="X-API-Key Header">
    ```bash
    curl -X GET "https://api.geogen.io/v1/entities" \
      -H "X-API-Key: wsk_your_api_key_here"
    ```
  </Tab>
</Tabs>

### API Key Format

All API keys follow this format:

```
wsk_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

- **Prefix:** `wsk_` (Workspace Key)
- **Body:** 32 random alphanumeric characters

### Authentication Errors

If authentication fails, you'll receive a `401 Unauthorized` response:

<CodeGroup>
```json Missing API Key
{
  "error": "Missing API key. Provide via Authorization: Bearer <key> or X-API-Key header."
}
```

```json Invalid API Key
{
  "error": "Invalid API key"
}
```

```json Deactivated Key
{
  "error": "API key is deactivated"
}
```
</CodeGroup>

### Example: Python

```python
import requests

API_KEY = "wsk_your_api_key_here"
BASE_URL = "https://api.geogen.io/v1"

headers = {
    "Authorization": f"Bearer {API_KEY}",
    "Content-Type": "application/json"
}

response = requests.get(f"{BASE_URL}/entities", headers=headers)
data = response.json()

print(data)
```

### Example: JavaScript/Node.js

```javascript
const API_KEY = process.env.GEOGEN_API_KEY;
const BASE_URL = 'https://api.geogen.io/v1';

const response = await fetch(`${BASE_URL}/entities`, {
  headers: {
    'Authorization': `Bearer ${API_KEY}`,
    'Content-Type': 'application/json'
  }
});

const data = await response.json();
console.log(data);
```

### Example: cURL

```bash
# Using environment variable
export GEOGEN_API_KEY="wsk_your_api_key_here"

curl -X GET "https://api.geogen.io/v1/entities" \
  -H "Authorization: Bearer $GEOGEN_API_KEY" \
  -H "Content-Type: application/json"
```
