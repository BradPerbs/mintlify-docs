---
title: "Get Workspace"
openapi: "GET /v1/workspace"
---

Get workspace information, usage limits, and credit balance.

<RequestExample>

```bash cURL
curl -X GET "https://api.geogen.io/v1/workspace" \
  -H "Authorization: Bearer wsk_your_api_key"
```


```javascript JavaScript
const response = await fetch('https://api.geogen.io/v1/workspace', {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY'
  }
});
const data = await response.json();
```


```python Python
import requests

response = requests.get(
    'https://api.geogen.io/v1/workspace',
    headers={'Authorization': 'Bearer YOUR_API_KEY'}
)
data = response.json()
```
