# Responses Tab

Explore Responses for additional insights.

![Responses Tab](https://i.imgur.com/1UNs1Uv.gif)

## Filtering Responses

In the **Responses** tab you can filter responses by **Mentioned/Not Mentioned**, by timeframe, by model and by tag.

## Exploring Additional Insights

By clicking on a response you can select entities cited within the response and they get highlighted within the response content, plus you can view the citations the AI model used to provide the recommendations for that particular response.
